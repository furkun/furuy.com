Bu uygulama ile kişisel verilerin korunmasının geliştirilmesi amaçlanıyor. Bu uygulama ile birlikte de, kişisel veri toplayan taraflara bazı sorumluluklar düşüyor. Web sitenizde birçok birinci parti ve/veya üçüncü parti çerezler kullanılıyor olabilir.
